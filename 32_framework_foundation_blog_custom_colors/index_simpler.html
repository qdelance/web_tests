Test using Foundation templates

Feb. 17 2015.

Designing a blog layout is a common use case, Foundation help building it by proving ready to use templates.

This blog post will show you how to use such a template.

Just use a default HTML template, loading Foundation JS and CSS.

Then all you have to do is select the proper template provided by Zurb and paste it in the body of your HTML template .
